建立个人博客

发表于 2020-03-19

1.安装node.js

去官网下载：https://nodejs.org/en/

一直next，修改存放地址，版本选第三个

安装完后进行测试

在桌面打开cmd，对node和npm进行测试

node -v

npm -v

两个都有版本号，就安装成功

2.配置node

修改npm的缓存目录和全局目录路径，将对应的模块目录改到D盘nodejs的安装目录。

在D盘的node的目录下创建两个目录

因为当npm安装的模块越来越多，那么这个文件夹的体积会越来越大，直到占满你的C盘。

cmd，代码

npm config set prefix “D:\nodejs\node_global” //配置全局模块存放路径
npm config set cache “D:\nodejs\node_cache” //cache路径

在电脑配置环境变量

测试配置是否成功：打开cmd，输入node，回车，再输入require(‘cluster’)，如果能正常输出cluster模块的信息，说明上面的所有配置就算生效了

3.安装hexo

先在cmd，阿里的国内镜像进行加速

在自己目录下，D:\Blog\Hexo

打开git bush here

npm i hexo-cli -g 安装hexo

hexo -v 验证版本号

hexo init 初始化文件

npm install 安装必备的组件

hexo g 生成静态网页

hexo s 打开本地服务器

浏览器打开 http://localhost:4000

Ctrl+C，关闭服务器

4.配置github

在目录下，D:\Blog\Hexo

打开git bush here

打开D:\Blog\Hexo下的_config.yml

修改最后一行

5.上传博客

打开git bush here

npm install hexo-deployer-git –save

hexo g

hexo d

出现Deploy done：git，表示成功，可以登录

https://XXXXXXX.github.io/

没有登录成功，消除缓存，等待响应
